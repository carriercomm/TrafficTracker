TrafficTracker

This site is created to show outgoing traffic from PFsense Debian, or basicly any platform you can get Node.js and WebSockets running. When done, you would see your outgoing traffic in realtime, and located on the map using Google Maps Api.
This site uses Bootstrap as a template.

Request from browser to start tshark -> websocket + js -> Node.js "Start tshark" -> tshark -> output via Node.js + websocket -> js -> geolocation lookup with Geoplugin -> input location data do Google Maps (latitude, longitude)
